# 架构文档 — 后端（Python FastAPI）

**项目：** 临床辅助诊断智能体（IDA MVP）
**部分：** backend
**生成日期：** 2026-03-16

---

## 执行摘要

后端是本系统的核心业务逻辑层，实现 IDA（缺铁性贫血）临床辅助诊断的"计算链路 + 检索链路 + 融合决策"三层架构。MVP 阶段重点实现计算链路（纯规则引擎），LLM 检索链路通过 Anthropic SDK 调用。

**核心设计原则：可解释、可审计、可复制。**

## 计算链路设计

### 诊断逻辑流程

```
输入 JSON
    │
    ▼
前置条件判断（precondition）
    ├── True → "不适用后续流程"
    └── False
        ▼
    Hb < 参考下限？（WS/T 405，按性别/孕期）
        ├── False → "不满足贫血诊断"（附干扰因素提示）
        └── True → 进入形态学分类
            ▼
        MCV < 80 AND MCH < 27 AND MCHC < 0.32？
            ├── False → 特别提示路径
            └── True
                ▼
            条款 2~9 中阳性数 ≥ 2？
                ├── True → "确诊 IDA"（附阳性条款明细）
                └── False → 特别提示路径
                    ▼
                SF < 14 μg/L？
                    ├── True → "高度提示 IDA"
                    └── False/None → "证据不足，不能诊断"
```

### 条款 0~9 说明

| 条款 | 判断内容 | 缺省行为 |
|------|----------|----------|
| 0 | Hb < 参考下限（按性别/孕期） | 必须提供 |
| 1 | 小细胞低色素（MCV/MCH/MCHC 均低） | 必须提供 |
| 2 | 明确缺铁病因 且 有临床体征 | 无则 c2=None |
| 3 | 血清铁 < 8.95 μmol/L 且 TIBC > 64.44 μmol/L | 未检查则跳过 |
| 4 | 转铁蛋白饱和度 TS < 0.15 | 未检查则跳过 |
| 5 | 骨髓铁染色阳性 | 未检查则跳过 |
| 6 | FEP 或 ZPP 升高 | 未检查则跳过（非计入 None） |
| 7 | SF < 14 μg/L（高权重，特别提示路径用） | 未检查则 c7=None |
| 8 | sTfR 升高（设备依赖） | 未开展则跳过 |
| 9 | 铁剂治疗后 Hb 升高 > 15 g/L | 未实施则跳过 |

**计分规则：** None（未检查）→ 不计分不扣分；True → 阳性 +1；False → 不计分

## 数据模型概览

### 输入（PatientInput）

```json
{
  "patient": {
    "sex": "female",          // 必须
    "age": 35,                // 必须
    "pregnant": false,        // 女性必须
    "precondition": false     // 必须
  },
  "lab": {
    "Hb": 85,                 // 必须（g/L）
    "MCV": 72,                // 必须（fL）
    "MCH": 24,                // 必须（pg）
    "MCHC": 0.30,             // 必须（0~1）
    "SF": 8,                  // 必须（μg/L）
    "TS": 0.09,               // 必须（0~1）
    // ... 可选字段
  },
  "clinical": {
    "etiology": { "chronic_bleeding": true },
    "symptoms": { "fatigue": true, "pallor": true }
  }
}
```

### 输出（DiagnosisOutput）

```json
{
  "conclusion": "可诊断为缺铁性贫血（IDA）",
  "level": "确诊",
  "severity": "中度",
  "stage": "缺铁性贫血期（IDA）",
  "evidence_detail": { "0": "...", "1": "...", "7": "..." },
  "differential_diagnosis": [...],
  "supplementary_tests": [...],
  "side_findings": [...],
  "remarks": [],
  "llm_description": "（LLM 生成，≤300 字）",
  "reference_guidelines": ["WS/T 405-2012"]
}
```

## 融合决策逻辑

| 条件 | 置信度 | 输出行为 |
|------|--------|----------|
| 计算链路 = IDA **且** 检索 Top-1 = IDA | 高 | 输出 IDA 诊断，附分期和程度 |
| 计算链路有冲突 **或** 检索 Top-1 不一致 | 低 | 降级，需人工复核，列出分歧点 |
| 关键指标缺失 | 低 | 标记缺失项，降级置信度 |

## 测试策略

- **单元测试**（pytest）：`tests/test_engine.py`
  - 确诊路径：小细胞低色素 + ≥2 条阳性
  - 提示路径：SF < 14 μg/L
  - 阴性路径：证据不足
  - 不适用路径：precondition = True
  - 边界条件：指标缺失（None 处理）
- **真实病例验证**（M2 里程碑）：10~20 个真实病例，覆盖率 ≥ 95%
- **临床医生验证**（M3 里程碑）：检验科医生金标准一致率 ≥ 85%
