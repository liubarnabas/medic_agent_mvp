## ida-diagnostic-mvp.jsx — 主组件说明

该组件是 MVP 阶段的核心 UI，预期包含以下功能区：

### 输入区
- 患者基本信息：性别、年龄、是否妊娠、前置条件标志
- 血常规指标：Hb、MCV、MCH、MCHC、RDW
- 铁代谢指标：SF、血清铁、TIBC、TS
- 炎症指标：CRP（可选）
- 临床信息：病因列表、症状/体征列表

### 结果展示区
- 诊断结论与等级（确诊/高度提示/阴性/不适用）
- 阳性证据明细（逐条）
- 贫血程度与分期
- 鉴别诊断建议
- 补充检查建议
- LLM 生成的自然语言诊断描述（≤300 字）
