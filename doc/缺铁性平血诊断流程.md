# 缺铁性贫血诊断逻辑

整个系统的输出包括：结构化诊断结论，以及大语言模型根据诊断依据生成的自然语言诊断描述。

## 缺铁性贫血诊断逻辑（文字版伪代码·供医生阅读）

> 说明：本节与下方 Python 伪代码逻辑完全一致，改用自然语言表述，便于临床理解与核对。

### 第一步：判断是否存在贫血（条款 0）

**前置条件判断：**

- 如果**满足**以下任一情况，不适用后续流程，直接跳到最终输出，建议去医院找专业医生判断：
  - 血容量变化：妊娠、肝硬化、低蛋白血症、充血性心力衰竭、脱水、大剂量利尿剂、急性大出血（如外伤、消化道大出血等）

- 如果**不满足**以上情况，则开始 **Hb 判断**：

  检查血红蛋白（Hb）是否低于国家标准参考下限（依据 WS/T 405 最新版本，系统每年自动更新）。

  - **若 Hb ≥ 参考下限** → 当前不满足贫血诊断，**停止**，并提示医生排除以下干扰因素后重新人工评估：
    - 高海拔地区长期居住（≥ 6～12 个月）
    - 长期慢性呼吸道疾病（如慢阻肺、长期吸烟、呼吸睡眠暂停等）
    - 影响 Hb 检测结果的其他因素

    > 以上情况不适用参考区间，故系统无法自动判断。

  - **若 Hb < 参考下限** → 确认存在贫血，进入第二步。

---

### 第二步：判断是否为小细胞低色素性贫血（条款 1）

同时满足以下三项：

| 指标 | 判断条件 |
|------|----------|
| MCV（平均红细胞体积） | < 参考下限（通常 80 fL） |
| MCH（平均红细胞血红蛋白含量） | < 参考下限（通常 27 pg） |
| MCHC（平均红细胞血红蛋白浓度） | < 参考下限（通常 0.32） |

- **不满足** → 非小细胞低色素性贫血，转入**特别提示路径**（见第四步）。
- **满足** → 进入第三步，累计支持证据。

---

### 第三步：累计支持证据（条款 2～9，满足 ≥ 2 条即可确诊）

逐一检查下列条款，每满足一条记 1 分；未检查/未开展的项目不计分、不扣分。

| 条款 | 判断内容 | 备注 |
|------|----------|------|
| **条款 2** | 有明确缺铁**病因**（至少 1 项）且有**临床体征**（至少 1 项） | 病因：吸收不良、素食/偏食、青春期发育、妊娠、多次妊娠、慢性出血或**其他病因**等；体征：黑便（出血）、 经血过多（出血）、慢性腹泻或者便秘、疲劳、倦怠、乏力、皮肤、粘膜和甲床苍白、心动过速、心悸、心绞痛、头晕目眩、注意力下降、呼吸加深，运动和情绪激动时更明显、反甲/勺状指（指甲薄脆、中央凹陷）、皮肤干燥、口角炎、舌炎、头发干枯易断、偏食、异食癖 | 
| **条款 3** | 血清铁 < 8.95 μmol/L **且** 总铁结合力 > 64.44 μmol/L | 注意单位换算：50 μg/dL ≈ 8.95 μmol/L；360 μg/dL ≈ 64.44 μmol/L |
| **条款 4** | 转铁蛋白饱和度 < 0.15（即 < 15%） | — |
| **条款 5** | 骨髓铁染色：外铁 −～±（阴性～弱阳） **且** 铁粒幼红细胞 < 15% | 部分医院开展；未检查则跳过 |
| **条款 6** | FEP > 0.9 μmol/L **或** ZPP > 0.96 μmol/L **或** ZPP > 3.0 μg/g Hb | 单位不同需换算 |
| **条款 7** | 血清铁蛋白 SF < 14 μg/L | ⚠️ 单独高权重，见第四步 |
| **条款 8** | sTfR > 26.5 nmol/L（R&D systems；其他设备界限值不同） | 部分医院开展；未开展则跳过 |
| **条款 9** | 铁剂治疗 2～4 周后 Hb 升高 > 15 g/L | 未实施则跳过 |

**计分后判断：**

- **条款 2～9 中满足 ≥ 2 条** → **确诊缺铁性贫血（IDA）**，同时输出所有阳性条款作为依据。
- **满足 < 2 条** → 进入第四步。

---

### 第四步：特别提示路径（条款 7 单独判断）

无论第二步结论如何（是否小细胞低色素），只要：

- **SF < 14 μg/L（条款 7 阳性）** → 输出提示：  
  *"血清铁蛋白 < 14 μg/L，高度提示缺铁性贫血，请结合其他检查进一步确认。"*

- **SF ≥ 14 μg/L（条款 7 阴性）或 SF 为空值** → 输出结论：  
  *"证据不足，不能诊断缺铁性贫血。建议补充检查 / 随访 / 试验性铁剂治疗后评估。"*

---

### 诊断路径总览

```
是否满足「前置条件」？（血容量变化：妊娠、肝硬化、低蛋白血症、充血性心力衰竭、脱水、大剂量利尿剂、急性大出血等）
├── 是 → 不适用后续流程，建议去医院找专业医生判断
└── 否
    └── Hb < 参考下限？（依据 WS/T 405 最新版本，系统每年自动更新）
        ├── 否（Hb ≥ 参考下限）→ 停止，当前不满足贫血诊断；提示排除干扰因素后重新人工评估
        │   （以下情况不适用参考区间：高海拔地区长期居住（≥ 6～12 个月）、
        │    长期慢性呼吸道疾病（如慢阻肺、长期吸烟、呼吸睡眠暂停等）、
        │    影响 Hb 检测结果的其他因素）
        └── 是（存在贫血）→ 进入第二步
            ├── MCV & MCH & MCHC 均低？（小细胞低色素）
            │   ├── 否 → 跳至【特别提示路径】
            │   └── 是
            │       └── 条款 2～9 满足 ≥ 2 条？
            │           ├── 是 → ✅ 确诊缺铁性贫血（IDA）
            │           └── 否 → 跳至【特别提示路径】
            └── 【特别提示路径】
                ├── SF 有值且 SF < 14 μg/L？
                │   ├── 是 → ⚠️ 高度提示 IDA，请进一步确认
                │   └── 否（SF ≥ 14 μg/L 或 SF 为空值）→ ❌ 证据不足，不能诊断缺铁性贫血；
                │       建议补充检查 / 随访 / 试验性铁剂治疗后评估
```

### 最终输出格式

每次诊断的最终输出包含两部分：

| 字段 | 内容 |
|------|------|
| **诊断结论** | 不适用（存在前置条件）/ 不满足贫血诊断 / 确诊 IDA / 高度提示 IDA / 证据不足，附诊断等级（确诊 / 提示 / 阴性） |
| **阳性证据明细** | 逐条列出满足的条款编号、检验值及所用参考标准年份 |
| **备注** | 需排除的干扰因素、未检查项目提示等 |
| **LLM 诊断描述** | 大语言模型根据以上结构化数据，用自然语言撰写的面向医生的诊断说明（≤ 300 字，含结论、阳性指标数值、参考标准、未尽事项）。 |


## 缺铁性贫血诊断逻辑（Python 伪代码·完整版）

> 说明：  
> - 所有阈值均以国家标准参考区间为准（见条款 0 注释），单位不同时需先换算再比较。  
> - 条款 2 中"其他病因"在系统初始化时由 Agent 检索循证证据后补全，用户可手动确认/扩充。  
> - 条款 8（sTfR）为部分医院开展项目，界限值与设备有关，未录入时跳过。  
> - 整个模型输出时需同步给出每条判断的依据（检验值 + 使用的参考标准年份）。
> - 最终输出包括结构化诊断结论与阳性证据明细，以及大语言模型根据诊断依据生成的自然语言诊断描述。

```python
# ============================================================
# 前置：参考区间（每年从 WS/T 405 最新版本自动更新，或手动触发）
# 参考来源：https://www.nhc.gov.cn/wjw/s9492/wsbz.shtml  搜索 WS/T 405
# 示例值来自 WS/T 405-2012，实际以系统加载的最新版为准
# ============================================================
REF = {
    "Hb_lower":   {"male": 120, "female": 110, "unit": "g/L"},   # 血红蛋白参考下限
    "MCV_lower":  80,    # fL
    "MCH_lower":  27,    # pg
    "MCHC_lower": 0.32,  # 或 320 g/L，视检验报告单位而定
}
REF_YEAR = "WS/T 405-2012"   # 系统初始化时自动填入最新年份

# ============================================================
# 输入：患者检验值与临床信息（由调用方传入）
# ============================================================
# lab["Hb"]      血红蛋白 (g/L)
# lab["MCV"]     平均红细胞体积 (fL)
# lab["MCH"]     平均红细胞血红蛋白含量 (pg)
# lab["MCHC"]    平均红细胞血红蛋白浓度 (0~1 或 g/L，统一换算)
# lab["serum_iron"]    血清铁 (μmol/L)；50 μg/dL ≈ 8.95 μmol/L
# lab["TIBC"]    总铁结合力 (μmol/L)；360 μg/dL ≈ 64.44 μmol/L
# lab["TS"]      转铁蛋白饱和度 (0~1)；15% → 0.15
# lab["SF"]      血清铁蛋白 (μg/L)
# lab["FEP"]     红细胞游离原卟啉 (μmol/L)；50 μg/dL ≈ 0.9 μmol/L
# lab["ZPP"]     血液锌原卟啉 (μmol/L 或 μg/g Hb)
# lab["sTfR"]    可溶性转铁蛋白受体 (nmol/L)；2.25 mg/L ≈ 26.5 nmol/L（R&D systems）
#                sTfR_available = False 表示本院未开展，跳过条款 8
# lab["Hb_after_iron_therapy"]   铁剂治疗 2~4 周后血红蛋白 (g/L)，未治疗则为 None
# clinical["sex"]   患者性别 "male" / "female"
# clinical["etiology"]    缺铁病因列表，至少一项为 True 表示有明确病因
#     可选值（系统初始化时由 Agent 检索后补全）：
#       "absorption_disorder"       吸收不良（胃/十二指肠/空肠近端手术或疾病）
#       "vegetarian_picky_eating"   素食/偏食
#       "adolescent_growth"         生长发育期（青春期）
#       "pregnancy"                 妊娠期
#       "multiparity"               多次妊娠（≥3 胎）
#       "chronic_bleeding"          慢性出血（消化道出血、月经过多等）
#       "other_evidenced"           其他病因（系统初始化时由 Agent 检索循证证据后补全）
# clinical["symptoms"]    临床体征列表，至少一项为 True 表示有临床体征
#     可选值：
#       "melena"                黑便（出血）
#       "menorrhagia"           经血过多（出血）
#       "chronic_diarrhea_constipation"  慢性腹泻或便秘
#       "fatigue"               疲劳、倦怠、乏力
#       "pallor"                皮肤、黏膜和甲床苍白
#       "tachycardia_etc"       心动过速、心悸、心绞痛、头晕目眩、注意力下降
#       "dyspnea_on_exertion"   呼吸加深（运动/情绪激动时更明显）
#       "koilonychia"           反甲/勺状指（指甲薄脆、中央凹陷）
#       "dry_skin"              皮肤干燥
#       "angular_cheilitis"     口角炎、舌炎
#       "brittle_hair"          头发干枯易断
#       "pica"                  偏食、异食癖
# lab["bone_marrow_iron_stain"]  骨髓铁染色结果
#     "negative_or_trace"  外铁 -~±（阴性~弱阳），铁粒幼红细胞 < 15%  → True
#     否则 → False
# clinical["precondition"]  前置条件标志（布尔值）
#     True 表示存在以下任一情况，不适用后续自动诊断流程：
#       妊娠、肝硬化、低蛋白血症、充血性心力衰竭、脱水、
#       大剂量利尿剂、急性大出血（如外伤、消化道大出血等）
#     → 直接输出"不适用"，建议去医院找专业医生判断

# ============================================================
# 各条款判定
# ============================================================

# 条款 0：贫血（Hb 低于参考下限）
c0 = lab["Hb"] < REF["Hb_lower"][clinical["sex"]]
evidence[0] = f"Hb={lab['Hb']} g/L，参考下限={REF['Hb_lower'][clinical['sex']]} g/L（{REF_YEAR}）"

# 条款 1：小细胞低色素贫血
c1 = (
    lab["MCV"]  < REF["MCV_lower"] and
    lab["MCH"]  < REF["MCH_lower"] and
    lab["MCHC"] < REF["MCHC_lower"]
)
evidence[1] = (
    f"MCV={lab['MCV']} fL（<{REF['MCV_lower']}），"
    f"MCH={lab['MCH']} pg（<{REF['MCH_lower']}），"
    f"MCHC={lab['MCHC']}（<{REF['MCHC_lower']}）"
)

# 条款 2：明确缺铁病因 且 有临床体征
has_etiology = any(clinical["etiology"].values())
has_symptom  = any(clinical["symptoms"].values())
c2 = has_etiology and has_symptom
evidence[2] = (
    f"病因：{[k for k,v in clinical['etiology'].items() if v]}；"
    f"体征：{[k for k,v in clinical['symptoms'].items() if v]}"
)

# 条款 3：血清铁 & 总铁结合力
c3 = lab["serum_iron"] < 8.95 and lab["TIBC"] > 64.44
evidence[3] = f"血清铁={lab['serum_iron']} μmol/L，TIBC={lab['TIBC']} μmol/L"

# 条款 4：转铁蛋白饱和度
c4 = lab["TS"] < 0.15
evidence[4] = f"转铁蛋白饱和度={lab['TS']:.2%}"

# 条款 5：骨髓铁染色（部分医院开展；未检查时值为 None，跳过）
if lab.get("bone_marrow_iron_stain") is not None:
    c5 = lab["bone_marrow_iron_stain"] == "negative_or_trace"
    evidence[5] = f"骨髓铁染色：{'外铁 -~±，铁粒幼红细胞<15%' if c5 else '未达标准'}"
else:
    c5 = None   # 未检查，不计入阳性数
    evidence[5] = "骨髓铁染色：未检查，跳过"

# 条款 6：FEP / ZPP（任一满足即为 True）
c6 = (
    lab.get("FEP",  0) > 0.9  or   # μmol/L；50 μg/dL ≈ 0.9 μmol/L
    lab.get("ZPP_umol", 0) > 0.96  or   # μmol/L；60 μg/dL ≈ 0.96 μmol/L
    lab.get("ZPP_ug_per_gHb", 0) > 3.0  # μg/g Hb
)
evidence[6] = (
    f"FEP={lab.get('FEP','—')} μmol/L，"
    f"ZPP={lab.get('ZPP_umol','—')} μmol/L / {lab.get('ZPP_ug_per_gHb','—')} μg/gHb"
)

# 条款 7：血清铁蛋白（高敏感度/特异度，单独提示；SF 为空值时跳过）
if lab.get("SF") is not None:
    c7 = lab["SF"] < 14
    evidence[7] = f"SF={lab['SF']} μg/L（参考阈值 <14 μg/L）"
else:
    c7 = None
    evidence[7] = "SF：未检查或未录入，跳过"

# 条款 8：sTfR（部分医院；界限值与设备有关，未开展时跳过）
if lab.get("sTfR_available", False):
    stfr_cutoff = lab.get("sTfR_cutoff", 26.5)   # nmol/L，系统初始化时手动设置
    c8 = lab["sTfR"] > stfr_cutoff
    evidence[8] = f"sTfR={lab['sTfR']} nmol/L（本院界限值={stfr_cutoff} nmol/L）"
else:
    c8 = None
    evidence[8] = "sTfR：本院未开展，跳过"

# 条款 9：铁剂治疗有效（治疗 2~4 周后 Hb 升高 >15 g/L）
if lab.get("Hb_after_iron_therapy") is not None:
    c9 = (lab["Hb_after_iron_therapy"] - lab["Hb"]) > 15
    evidence[9] = (
        f"治疗后 Hb={lab['Hb_after_iron_therapy']} g/L，"
        f"较基线升高 {lab['Hb_after_iron_therapy'] - lab['Hb']} g/L"
    )
else:
    c9 = None
    evidence[9] = "铁剂治疗：尚未实施或结果未录入，跳过"

# ============================================================
# 主判断逻辑 → 生成结构化结论 + LLM 诊断描述
# ============================================================

# 条款 2~9 中阳性数量（None 表示未检查，不计入）
supporting_conditions = [c2, c3, c4, c5, c6, c7, c8, c9]
positive_count = sum(1 for c in supporting_conditions if c is True)

# ---------- 第一部分：结构化结论 ----------

# 第〇步：前置条件判断（血容量变化等 → 不适用自动诊断）
if clinical.get("precondition", False):
    conclusion = {
        "verdict":         "不适用后续流程",
        "level":           "不适用",
        "active_evidence": {},
        "remarks": [
            "存在血容量变化等前置条件（妊娠、肝硬化、低蛋白血症、充血性心力衰竭、"
            "脱水、大剂量利尿剂、急性大出血等），建议去医院找专业医生判断",
        ],
    }

# 第一步：Hb 未低于参考下限 → 不满足贫血诊断
elif not c0:
    conclusion = {
        "verdict":         "不满足贫血诊断",
        "level":           "阴性",
        "active_evidence": {0: evidence[0]},
        "remarks": [
            "提示医生排除以下干扰因素后重新人工评估：",
            "· 高海拔地区长期居住（≥ 6～12 个月）",
            "· 长期慢性呼吸道疾病（如慢阻肺、长期吸烟、呼吸睡眠暂停等）",
            "· 影响 Hb 检测结果的其他因素",
            "以上情况不适用参考区间，故系统无法自动判断。",
        ],
    }

# 第二步 + 第三步：小细胞低色素 且 条款 2～9 满足 ≥ 2 条 → 确诊 IDA
elif c1 and positive_count >= 2:
    active_keys = [0, 1] + [i + 2 for i, c in enumerate(supporting_conditions) if c is True]
    conclusion = {
        "verdict":         "可诊断为缺铁性贫血（IDA）",
        "level":           "确诊",
        "active_evidence": {k: evidence[k] for k in active_keys},
        "remarks":         [],
    }

# 第四步：特别提示路径（条款 7 单独判断）
else:
    if c7 is True:
        conclusion = {
            "verdict":         "高度提示缺铁性贫血，尚不满足确诊标准",
            "level":           "提示",
            "active_evidence": {7: evidence[7]},
            "remarks":         ["SF < 14 μg/L 敏感度/特异度较高，请结合其他检查进一步确认"],
        }
    else:   # c7 is False（SF ≥ 14）或 c7 is None（SF 为空值）
        conclusion = {
            "verdict":         "证据不足，不能诊断缺铁性贫血",
            "level":           "阴性",
            "active_evidence": {k: evidence[k] for k in range(10) if evidence.get(k)},
            "remarks":         ["建议补充检查 / 随访 / 试验性铁剂治疗后评估"],
        }
```
